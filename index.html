Time Management

  • First 5 minutes: Read ALL tasks quickly
  • Easy tasks first: Users, firewall, basic mounts
  • Hard tasks last: Troubleshooting, complex LVM

Must-Know Commands

nmcli con mod semanage fcontext setsebool -P lvextend -r firewall-cmd --permanent podman generate systemd chmod 2775 setfacl -m d:

Common Mistakes

  • Forgetting --permanent in firewall-cmd
  • Using chcon instead of semanage fcontext
  • Not testing with mount -a after editing fstab
  • Skipping reboot verification

Verification Steps

  1. Run systemctl --failed
  2. Check getenforce (must be Enforcing)
  3. Verify firewall-cmd --list-all
  4. Check all mounts with df -h
  5. REBOOT TEST is mandatory!
